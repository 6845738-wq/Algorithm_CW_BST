# Курсова робота з теорії алгоритмів

## Опис

Репозиторій містить програмний код до курсової роботи з дисципліни «Теорія алгоритмів».

У роботі виконано дві частини:

1. Побудова та аналіз двійкового дерева пошуку за варіантом №23.
2. Розробка інтерактивного візуалізатора алгоритму Кнута — Морріса — Пратта.

## Структура репозиторію

```text
Algorithm_CW_BST/
│
├── task1_bst_analysis.py
├── task2_kmp_visualizer.py
├── README.md
└── requirements.txt
```

## Перша частина

Файл `task1_bst_analysis.py` містить програму для побудови та аналізу двійкового дерева пошуку.

Вхідна послідовність варіанта №23:

```text
10, 20, 30, 40, 50, 9, 8, 7, 6, 11, 12, 13
```

Програма виконує:

- побудову дерева;
- симетричний обхід;
- обчислення кількості вузлів;
- обчислення висоти дерева;
- визначення найдовших шляхів.

Очікуваний симетричний обхід:

```text
6, 7, 8, 9, 10, 11, 12, 13, 20, 30, 40, 50
```

Висота дерева:

```text
4 ребра / 5 рівнів
```

## Друга частина

Файл `task2_kmp_visualizer.py` містить інтерактивний графічний візуалізатор алгоритму Кнута — Морріса — Пратта.

Контрольний приклад:

```text
Текст:   ABABDABACDABABCABAB
Шаблон: ABABCABAB
```

Для шаблону будується префікс-функція:

```text
0, 0, 1, 2, 0, 1, 2, 3, 4
```

Результат пошуку:

```text
Шаблон знайдено з позиції 10
```

Візуалізатор підтримує:

- введення тексту і шаблону;
- побудову префікс-функції;
- пошук шаблону в тексті;
- покроковий режим;
- автоматичний режим;
- регулювання швидкості;
- очищення результатів.

## Запуск

Потрібен Python 3.10 або новіший.

Запуск першої програми:

```bash
python task1_bst_analysis.py
```

Запуск візуалізатора КМП:

```bash
python task2_kmp_visualizer.py
```

Додаткові бібліотеки не потрібні, оскільки Tkinter входить до стандартної бібліотеки Python.
