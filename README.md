# Курсова робота з теорії алгоритмів

## Тема

Побудова, аналіз та візуалізація двійкового дерева пошуку для варіанта №23.

## Вхідні дані

Послідовність елементів для побудови двійкового дерева пошуку:

```text
10, 20, 30, 40, 50, 9, 8, 7, 6, 11, 12, 13
```

## Структура репозиторію

```text
Algorithm_CW_BST/
│
├── task1_bst_analysis.py
├── task2_bst_visualizer.py
├── README.md
└── screenshots/
```

## Опис файлів

### task1_bst_analysis.py

Консольна програма для побудови та аналізу двійкового дерева пошуку. Програма виконує:

- послідовну вставку елементів;
- симетричний обхід дерева;
- обчислення кількості вузлів;
- обчислення висоти дерева;
- виведення найдовших шляхів.

### task2_bst_visualizer.py

Графічний візуалізатор двійкового дерева пошуку на Python з використанням Tkinter. Програма підтримує:

- завантаження послідовності варіанта №23;
- ручне додавання елементів;
- пошук значень;
- покроковий режим;
- автоматичний режим;
- очищення дерева;
- відображення висоти, кількості вузлів та InOrder-обходу.

## Як запустити

Потрібен Python 3.10 або новіший.

Запуск першої програми:

```bash
python task1_bst_analysis.py
```

Запуск візуалізатора:

```bash
python task2_bst_visualizer.py
```

Додаткові бібліотеки не потрібні, оскільки Tkinter входить до стандартної бібліотеки Python.

## Очікуваний результат

Для варіанта №23 симетричний обхід дерева повинен мати вигляд:

```text
6, 7, 8, 9, 10, 11, 12, 13, 20, 30, 40, 50
```

Висота побудованого дерева:

```text
4 ребра / 5 рівнів
```
